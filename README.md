# 🔐 Live Location Tracking Awareness using Kali Linux

## 📌 Project Overview
Demonstrated how fake chatbots can be used to track a person's live location without their knowledge, raising awareness about cybersecurity threats and safe internet practices.

## 🛠️ Technologies Used
- Kali Linux
- Hound Tool
- Social Engineering Techniques

## 🔍 How It Works
- Created a fake chatbot that generates a malicious tracking link
- When the victim copies and opens the link, their live location is captured
- Demonstrated how attackers exploit curiosity and trust to track victims
- Educated audience on identifying and avoiding suspicious links

## 🎯 Key Highlights
- 🏆 Won 1st Prize at College Cyber Security Expo
- Raised awareness about fake chatbot threats
- Demonstrated real-world social engineering attack techniques
- Promoted safe internet practices and personal data protection
